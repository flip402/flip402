# Contributing

FLIP402 is a community experiment. Contributions welcome.

## What's useful

- Bug fixes
- Mobile UX improvements
- Provably fair randomness implementation (see `SECURITY.md`)
- Translations

## What's not needed

- New game modes
- Token mechanics
- Roadmap features

## How to contribute

1. Fork the repo
2. Create a branch: `git checkout -b fix/your-fix`
3. Make your changes
4. Open a PR with a clear description

## Code style

TypeScript strict mode. Tailwind for styles. Framer Motion for animations. Keep it simple.
